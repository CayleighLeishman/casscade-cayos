CUBE CSS

So Cayleigh, what exactly is this for?

Great question, other Cayleigh! CUBE stands for Composition, Utility, Block, Exception CSS. It's a methodology geared towards simplicity, pragmatism, and consistency.

It’s a way to think about how to structure your CSS in a way that is easy to maintain and understand.

It’s not a framework, but rather a set of principles that can be applied to any CSS project.

If that doesn't make sense, don't panic! As Brother Grimmet told us, learning anything new is supposed to be tricky. That's also a good thing because it means you're learning something new!
